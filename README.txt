These are the step to get app running 

commands
1.) Run this on a fresh install of a linux based operating system using a virtual box. This confirms that we have a untainted enviroment 

2.) run install script inside of folder. please run "chmod +x install.sh" if file is not already excutable 

3.) run terminal command:
    export FLASK_APP=server.py

4.) run the server with command: 
    flask run 
    
5.) visit http://127.0.0.1:5000/
